Luyện tập sử dụng các thao tác cơ bản với Git

Mô tả

Trong phần này, chúng ta sẽ thực hiện các bước thao tác cơ bản nhất để bắt đầu làm việc với Git và GitHub. Bắt đầu từ việc tạo dự án trên GitHub cho đến việc đưa các file mã nguồn lên GitHub. Để hoàn thành bài tập, cần nộp link của dự án lên CodeGymX.
